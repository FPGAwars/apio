Experimenting with Apio aware MCP server. This folder is not part of Apio CLI and may be deleted with no further notice.

## Running the MCP server

This runs the server on port 8000

```
./apio-mcp-server.sh
```

## Running the MCP server inspector

```
./mcp-inspector.sh
```

## Connecting the inspector to the server

On the left side of the inspector page in the browser, select transport=`Streamable HTTP` and URL=`http://localhost:8000/mcp`.
